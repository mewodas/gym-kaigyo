# ジム開業で使える助成金・補助金 完全リスト

開業資金を**最大400〜500万円カット**できる助成金・補助金。本記事ではジム開業者が活用できる主要な制度を一覧で整理します。

## 助成金と補助金の違い

| 種別 | 助成金 | 補助金 |
|---|---|---|
| 主な目的 | 雇用関連 | 事業発展支援 |
| 主管 | 厚生労働省 | 経済産業省・自治体 |
| 採択率 | 高（条件を満たせば概ね通る） | 中〜低（審査あり） |
| 申請時期 | 通年 | 年数回の公募 |
| 上限額 | 数十万〜数百万 | 数十万〜数千万 |

**「ジム開業」で使えるのは主に補助金**。雇用関連の助成金は採用後に活用。

## 国の補助金（最重要）

### 1. 小規模事業者持続化補助金

| 項目 | 内容 |
|---|---|
| 上限額 | 50〜250万円 |
| 補助率 | 2/3〜3/4 |
| 対象 | 販路開拓、業務効率化 |
| 公募時期 | 年4回程度 |
| 採択率 | 40〜60% |

詳細は[小規模事業者持続化補助金の使い方](/hiyou/hojokin)で解説。

### 2. 事業再構築補助金

| 項目 | 内容 |

| 対象 | 業種転換・事業再構築 |
| 公募時期 | 年数回 |
| 採択率 | 30〜50% |

**異業種からの転身者**が活用可能。サラリーマンからジム経営への転身など。

### 3. ものづくり補助金

| 項目 | 内容 |
|---|---|
| 上限額 | 750〜3,000万円 |

| 対象 | 革新的サービス開発 |
| 公募時期 | 年4回程度 |
| 採択率 | 40〜60% |

**「AI活用ジム」「オンライン×対面ジム」**等、革新的なコンセプトなら対象。

### 4. IT導入補助金

| 項目 | 内容 |
|---|---|
| 上限額 | 30〜450万円 |
| 補助率 | 1/2〜2/3 |
| 対象 | IT導入（顧客管理ツール等） |
| 公募時期 | 通年（複数枠あり） |
| 採択率 | 40〜70% |

**Hacomono・ジムDX等の導入時**に使える。

### 5. 創業助成金（東京都）

| 項目 | 内容 |
|---|---|
| 上限額 | 400万円 |
| 補助率 | 2/3 |
| 対象 | 都内で創業5年以内の企業 |
| 公募時期 | 年2回 |
| 採択率 | 30〜40% |

**東京都で開業する方は必ずチェック**。

## 補助金活用の戦略

### 戦略1：複数申請の使い分け

```
開業前：日本政策金融公庫の融資（300万円）
開業時：小規模事業者持続化補助金（150万円）
開業後：IT導入補助金（顧客管理ツール導入：50万円）
半年後：事業再構築補助金（拡大時：500万円）
```

**段階的に活用**することで、合計1,000万円以上の支援を受けることも可能。

### 戦略2：商工会議所のサポート活用

- 申請書作成のアドバイス
- 採択率の高い書き方の指導
- 過去の採択事例の共有

**個人申請より採択率が15〜25%高い**。

### 戦略3：認定支援機関との連携

- 中小企業診断士・税理士・弁護士
- 申請書の添削・アドバイス
- 採択後の実績報告サポート

費用は5〜30万円程度。投資対効果が高い。

## 補助金申請の流れ（共通）

1. **公募要領を確認**
2. **事業計画書の作成**
3. **必要書類の準備**
4. **電子申請（jGrants等）**
5. **採択発表**（2〜3ヶ月後）
6. **補助事業実施**
7. **実績報告**
8. **補助金入金**（さらに2〜3ヶ月後）

**申請から入金まで6〜10ヶ月**。先払いができる手元資金が必要。

## 採択されやすい申請書のポイント

### 1. 数字で根拠を示す
- 「集客が伸びる」より「月10件→25件に増える」

### 2. 補助金で何を達成するか明確に
- 「○○への投資で売上を○%増やす」

### 3. 地域・業界への貢献
- 「地域の健康増進に貢献」
- 「雇用創出（将来的に2名）」

### 4. 競合分析・市場性
- データを使った市場分析
- 自社の差別化ポイント

### 5. 実現可能性
- 過去の経験・実績
- 段階的な実行計画

## 補助金の落とし穴

### ❌ 「採択前に発注」する
**採択日以降の発注のみ補助対象**。フライング購入は対象外。

### ❌ 領収書・契約書を紛失
実績報告で必要。**徹底管理**。

### ❌ 補助対象外の経費を計上
事前に対象経費を確認。

### ❌ 報告書の遅延
期限を1日でも過ぎると補助金没収のリスク。

### ❌ 事業継続義務違反
補助金には数年間の事業継続義務がある場合あり。

## まとめ

ジム開業で使える補助金・助成金は**最大400〜500万円のサポート**が可能。「持続化補助金（150万円）+ IT導入補助金（50万円）+ 自治体の創業助成金（200万円）」を組み合わせれば、開業費用の大半をカバーできます。商工会議所や認定支援機関のサポートを活用して採択率を高め、計画的に申請を進めましょう。
