# 鏡と床材で決まるジムの印象

ジムの第一印象を作るのが**鏡と床**。本記事では、これらの選び方を実例付きで解説します。

## ジムにおける鏡の役割

| 役割 | 重要度 |
|---|---|
| フォーム確認 | ★★★ |
| 空間を広く見せる | ★★★ |
| 顧客の自信向上 | ★★ |
| ジムらしい雰囲気作り | ★★ |

**鏡は単なる装飾ではなく、サービス品質に直結**します。

## 鏡の種類と特徴

### 1. クリスタル鏡（業務用標準）

| 項目 | 内容 |
|---|---|
| 価格 | 30,000〜100,000円/㎡ |
| 厚み | 5〜8mm |
| 反射率 | 92〜95% |
| 耐久性 | 高 |
| 歪み | 最小 |

**ジムで一般的に使われる業務用標準**。

### 2. 普通の鏡（住宅用）

| 項目 | 内容 |
|---|---|
| 価格 | 5,000〜30,000円/㎡ |
| 反射率 | 85〜90% |
| 歪み | あり（フォーム確認に難） |

**コスト最重視の場合のみ**。フォーム確認には向かない。

### 3. アクリルミラー

| 項目 | 内容 |
|---|---|
| 価格 | 中程度 |
| メリット | 軽量・割れにくい |
| デメリット | 傷がつきやすい・歪みあり |

**子どもがいるジム・グループレッスン向き**。

### 4. フィルムタイプ（簡易設置）

| 項目 | 内容 |
|---|---|
| 価格 | 1,000〜5,000円/㎡ |
| メリット | 既存壁にすぐ貼れる |
| デメリット | 反射率・耐久性が低い |

**仮設や試しに導入**したい時のみ。

## 鏡の必要面積

### 推奨設置範囲

| ジムタイプ | 必要鏡面積 |
|---|---|
| 個人パーソナル（10坪） | 4〜8㎡ |
| 中規模パーソナル（20坪） | 10〜15㎡ |
| グループレッスン対応 | 全壁面の70%以上 |

**最低でも壁1面（4〜6㎡）**は必要。

### 設置位置のセオリー

1. **トレーニングエリアの正面**：フォーム確認用
2. **トレーニングエリアの側面**：横からのフォーム確認
3. **入口付近**：空間を広く見せる視覚効果

**正面と側面の2方向以上**にあると顧客満足度が高い。

## 鏡設置の費用相場

### 業務用鏡（クリスタル鏡）

| 設置面積 | 材料費 | 工事費 | 合計 |
|---|---|---|---|
| 4㎡（壁1面） | 12〜32万円 | 5〜10万円 | 17〜42万円 |
| 8㎡（壁2面） | 24〜64万円 | 10〜20万円 | 34〜84万円 |
| 15㎡（壁3面） | 45〜120万円 | 20〜30万円 | 65〜150万円 |

**鏡の予算は内装工事の10〜20%**を占めることが多い。

## 鏡選びの注意点

### サイズと厚み
- 高さ200cm以上が望ましい
- 厚み6mm以上の業務用
- 端の処理（角を丸める or カバー）

### 安全対策
- 飛散防止フィルムを必ず貼る
- 地震時の落下防止アンカー
- 子ども向けエリアには低反射タイプ

### メンテナンス
- 専用クリーナーで月1回清掃
- 油・汗汚れに対応した素材

## ジムの床材の種類

### 1. ジムマット（タイル型）

| 項目 | 内容 |
|---|---|
| 価格 | 5,000〜15,000円/㎡ |
| 厚み | 10〜30mm |
| 防音性 | 高 |
| 耐久性 | 高 |

**ジムでは最も一般的**。タイル状で交換が容易。

### 2. ロール型ラバーマット

| 項目 | 内容 |
|---|---|
| 価格 | 5,000〜12,000円/㎡ |
| 厚み | 4〜8mm |
| メリット | 継ぎ目なしで美しい |
| デメリット | 設置が大変 |

**プロフェッショナルな見た目**を実現。

### 3. ジョイントマット（家庭用）

| 項目 | 内容 |
|---|---|
| 価格 | 1,500〜3,000円/㎡ |
| 厚み | 10〜20mm |
| メリット | 安い・自分で設置可 |
| デメリット | 業務利用には耐久性不足 |

**個人ホームジム向け**。商業利用は避ける。

### 4. リノリウム・PVC床

| 項目 | 内容 |
|---|---|
| 価格 | 3,000〜8,000円/㎡ |
| メリット | フラット・清潔感 |
| デメリット | 衝撃吸収力が低い |

**ストレッチエリアやウォームアップエリア向き**。

## 床材選びのポイント

### 衝撃吸収性
- ダンベル落下に耐えるか
- 階下への振動を抑えるか
- **20mm以上のジムマットが基本**

### 滑り止め性能
- 汗や水で滑らないか
- 表面のテクスチャー

### 清掃のしやすさ
- 水拭きできるか
- 汚れが目立たない色

### 耐久性
- 長期間の使用に耐えるか
- 業務用グレードか

## 床材の費用相場

### 標準的なパーソナルジム（15坪）

| 項目 | 費用 |
|---|---|
| ジムマット（材料） | 30〜80万円 |
| 防音層（必要時） | 10〜20万円 |
| 設置工事 | 5〜15万円 |
| **合計** | **45〜115万円** |

**床材は内装工事の20〜30%**を占めることが多い。

## エリア別の床材の使い分け

### トレーニングエリア
- 厚み20〜30mmのジムマット
- 高負荷器具周りはさらに補強

### ストレッチエリア
- 薄手のヨガマット床
- またはカーペット

### 受付・カウンセリングエリア
- フローリング風 or リノリウム
- 清潔感重視

### 更衣室・通路
- ノンスリップタイル
- 水濡れ対応

## 鏡と床のコーディネート

### 印象別の組み合わせ

| 印象 | 鏡 | 床 |
|---|---|---|
| プロフェッショナル | 大面積クリスタル鏡 | 黒・グレーのマット |
| 高級感 | アクセント鏡 | ダーク木目調 |
| 清潔・明るい | 全壁面鏡 | 白・ベージュ系 |
| カジュアル | 部分鏡 | カラフルなマット |

**ターゲット顧客の好み**に合わせて選定。

## 失敗例

### ❌ 失敗1：安い鏡で歪み
住宅用の鏡を使ったところ、フォームが正確に確認できず顧客クレーム。

### ❌ 失敗2：床の防音不足
階下に振動が響き、近隣からクレーム→大規模工事で再施工。

### ❌ 失敗3：床の滑り
コーティングが弱く、汗で滑って怪我事故。

### ❌ 失敗4：設置位置の失敗
鏡の位置が悪く、トレーニング中に確認しづらい。

### ❌ 失敗5：色の選択ミス
明るすぎる床で汚れが目立ち、清潔感を損なう。

## DIY vs プロ依頼

### DIYで可能なもの
- 簡易ジョイントマット
- フィルム鏡
- 部分的な補修

### プロ依頼が必要なもの
- 業務用クリスタル鏡の設置
- 大面積のジムマット
- 防音層を含む床工事

**プロ依頼の費用 vs 失敗のリスク**を比較して判断。

## まとめ

ジムの鏡と床は**「機能性 × 安全性 × 印象」**の3軸で選定。鏡はクリスタル鏡（壁1〜2面）、床は20mm以上のジムマットが基本セット。設置費用は内装の30〜50%を占める重要パートですが、ここの質がジムの印象と顧客満足度を大きく左右します。**安さではなく、5〜10年使える品質**を選びましょう。
