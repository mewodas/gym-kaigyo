# SNS投稿を月10分で2週間分作る方法

「SNS運用が時間を取りすぎる」——これがジム経営者の最大の悩みの一つ。**Claudeを使えば、月1セッション10分で2週間分の投稿が完成**します。本記事では実際に使っているワークフローを公開します。

## 自動化前後の比較

| 項目 | 手動運用 | AI活用後 |
|---|---|---|
| 投稿1本にかかる時間 | 30〜60分 | 1〜3分 |
| 月14本の投稿時間 | 7〜14時間 | 30〜60分 |
| キャプション作成 | 都度 | 月1回まとめて |
| ハッシュタグ選定 | 都度 | 月1回まとめて |
| 削減率 | — | **約90%削減** |

## 自動化の全体ワークフロー

```
月1回（30分）：投稿テーマ・キャプション・ハッシュタグをまとめて生成
   ↓
週1回（10分）：画像・動画を撮影
   ↓
当日（5分）：予約投稿ツールに登録
```

**実作業：月1時間以内**で運用できます。

## Step 1：月初の30分セッション

### 必要な情報

事前に以下を整理：

- 月のテーマ（例：「夏に向けたダイエット」）
- ターゲット顧客の悩み
- 直近のお客様の成果（許可済み）
- 季節のイベント・キャンペーン

### Claudeへの依頼プロンプト

```
パーソナルジムのInstagram投稿を14本作成してください。

【ジム情報】
- 場所：五反田駅徒歩5分
- ターゲット：30〜50代女性、ダイエット目的
- 強み：食事管理AI付きパーソナル指導

【今月のテーマ】
夏に向けたボディメイク準備

【投稿パターン（必ず以下を含める）】
- お客様のBefore/After紹介：3本
- トレーニング解説：4本
- 食事Tips：3本
- ジムの様子・雰囲気：2本
- 経営者の想い・ストーリー：2本

【各投稿に含めること】
1. キャプション本文（200〜300文字）
2. ハッシュタグ20個（大3：中5：小5に分けて）
3. 画像/動画のアイデア（撮影シーン）
4. 投稿の狙い（認知 / エンゲージ / CV）
```

### 出力例

Claudeから以下のような出力が返ってきます：

> **投稿#1：トレーニング解説**
>
> キャプション：
> 「お腹痩せの近道は腹筋じゃない！
> 実はヒップアップ系の種目が一番効きます。
> ダイエット成功の8割は『どこを鍛えるか』で決まります。
> 詳しい理由とおすすめ種目はストーリーズで解説中です。」
>
> ハッシュタグ：
> #ダイエット #五反田ジム #お腹痩せ #ヒップアップ ...
>
> 画像案：
> ヒップスラスト中の様子を真横から（横向き全身）
>
> 投稿の狙い：エンゲージメント向上

これを14本一気に生成。**所要時間：5〜10分**。

## Step 2：撮影スケジュール（週1回）

生成された投稿に必要な画像・動画を**まとめて週末に撮影**：

| 撮影項目 | 推奨時間 |
|---|---|
| Before/After用素材 | 顧客のセッション中（許可取得必須） |
| トレーニング解説動画 | 自分でやる、30秒×4本 |
| ジム雰囲気写真 | 開店前/閉店後の無人時間 |
| 食事写真 | 自分で食べる前に撮る |

**1週間分の素材を1回でまとめ撮りする**のが効率的。

## ストーリーズもパッケージ化

ストーリーズはストック投稿と違い、**24時間で消える気軽な発信**。

### 自動化フロー

1. 月初に「ストーリーズの型」を10パターン作る
2. その日の状況に合わせて適用
3. テキスト・スタンプを少し変えるだけで投稿

### ストーリーズの型（例）

- 朝のトレーニング様子
- 食事写真とコメント
- お客様との対話（許可済み）
- 質問BOXへの回答
- 今日のおすすめ食材

## ハッシュタグ管理

ハッシュタグは**カテゴリ別にストック**しておくと毎回の選定が不要に：

| カテゴリ | ストック数 |
|---|---|
| ダイエット系 | 30個 |
| トレーニング系 | 30個 |
| 食事系 | 20個 |
| 地域系 | 10個 |
| ライフスタイル系 | 20個 |

**Notionにテンプレ化**しておけばコピペで使えます。

## 月10分セッションの所要時間

| 作業 | 時間 |

| Claude にプロンプト送信 | 1分 |
| 出力を確認・微調整 | 5分 |
| 予約投稿ツールに登録 | 4分 |

これで**2週間分の投稿が完成**。

## 自動化の落とし穴

### ❌ 完全AI任せにしない
- 自分の体験・固有名詞・実数字を必ず加える
- 「機械的」と感じる投稿はエンゲージメント低下

### ❌ 全部同じパターンに偏る
- 14本すべてが「Before/After」だと飽きられる
- パターンの分散を必ず確認

### ❌ AI生成バレを警戒しすぎる
- 適切に編集すればバレない
- 投稿者の人格が出ていれば問題なし
