# AIを「経営参謀」として活用する方法

「数字は集めているけど、どう判断すればいいかわからない」——多くのジム経営者が抱える悩み。本記事では、AIに経営数字を渡して**「いま何をすべきか」を判断するための具体的なプロンプト**を公開します。

## なぜAIが経営判断に有効か

| 観点 | 人間 | AI |
|---|---|---|
| 客観性 | 感情が入る | 完全に客観的 |
| 計算スピード | 遅い | 即時 |
| 視点の広さ | 経験に依存 | 業界知識を網羅 |
| 提案の数 | 1〜3個 | 10個以上 |
| 時間コスト | 数時間 | 数分 |

**「経営判断のセカンドオピニオン」**としてAIは極めて優秀。

## 月次の振り返りに使うプロンプト

### マスタープロンプト

```
あなたはパーソナルジム経営の専門家です。
以下の数字を分析して、改善ポイントを5つ提案してください。

【今月の数字】
- 売上：◯◯円（前月比◯%）
- 新規入会：◯名
- 既存退会：◯名
- 体験予約：◯件
- 体験→入会率：◯%
- 紹介経由の新規：◯名
- レンタルジム稼働率：◯%
- 食事管理オプション加入率：◯%

【改善提案の条件】
- 具体的なアクション（誰が、いつ、何をする）
- 優先順位
- 期待される効果
- 実行コスト
```

### 出力例

> **改善ポイント1：体験→入会率の向上**
> 
> 現状：25%（前月35%）。体験後のフォローが不足している可能性。
> 
> アクション：体験当日の22:00までにLINEで丁寧なフォローメッセージを送る。3日後にもリマインド。
> 
> 期待効果：転換率35%への回復で月+2名・売上+8万円
> 
> 実行コスト：時間のみ（金銭ゼロ）

このような分析結果が**5項目分**返ってきます。

## 経営判断別のプロンプト集

### 1. 価格改定の判断

```
パーソナルジムの価格改定を検討しています。

【現在の価格】7,500円/h
【五反田エリア相場】8,000〜12,000円/h
【顧客数】10名（うち長期会員8名）
【月の稼働】65h
【月商】520,000円

【検討事項】
- 9,000円に上げるべきか
- 上げる場合のリスクと対策
- 既存客への伝え方
- 新規価格との段階的移行プラン
```

### 2. 採用の判断

```
業務委託トレーナーの採用を検討中。

【現状】
- 自分の稼働：月65h（ほぼフル）
- 月商：520,000円
- 新規予約を断り始めている
- 月45万円が3ヶ月連続

【質問】
- 採用すべきタイミングか？
- 業務委託 vs 雇用、どちらが適切か？
- 期待される増益効果
- リスクと対策
```

### 3. メニュー改定の判断

```
新メニューの追加を検討中。

【既存メニュー】
- スタンダード（60分）月4回：32,000円
- スタンダード（60分）月8回：60,000円
- ライト（30分）月4回：18,000円
- ライト（30分）月8回：33,600円

【追加候補】
- 食事管理オプション（+13,200円）
- AIフォーム解析オプション（+5,000円）
- オンラインコース（28,000円/月）

【顧客属性】
- 30〜50代女性中心、ダイエット目的

【判断】
- どのオプションを優先すべきか
- 価格設定の妥当性
- 期待される売上増効果
```

### 4. 集客チャネルの優先順位

```
広告予算月10万円の最適配分を提案してください。

【ジム情報】
- 五反田パーソナルジム
- ターゲット：30〜50代女性
- 月新規目標：5名
- 体験→入会率：50%

【チャネル候補】
- Instagram広告
- Google広告
- ホームページSEO
- ブログ運営
- Googleビジネスプロフィール（無料）
- 紹介制度（インセンティブ）
- 地域チラシ

【ROIと投資効率を考慮した最適配分を】
```

### 5. 退会防止策

```
退会率が高くなっています。原因分析と対策を提案してください。

【データ】
- 月退会率：15%（業界平均5〜10%）
- 退会理由トップ3：
  1. 効果を感じない（40%）
  2. 通うのが面倒（30%）
  3. 料金が高い（20%）
- 平均継続期間：3.5ヶ月（業界8〜12ヶ月）

【優先度の高い改善策5つを提案】
```

## AIに渡すデータの整理方法

### スプレッドシートで月次管理

#### 必須シート
1. **月次サマリー**：売上・客数・KPI
2. **顧客別**：個別の継続状況
3. **集客分析**：チャネル別の流入
4. **コスト**：固定費・変動費

### AIへ渡すフォーマット

#### Markdownテーブル形式
- AIが読みやすい
- 比較しやすい

#### CSVデータ
- 大量データに対応
- グラフ化の前段階

## 業界比較分析

```
私のジムの数字を業界平均と比較分析してください。

【私のジム】
- 月商：48万円
- 客単価：32,000円
- 継続率（3ヶ月）：75%
- LTV：約10万円
- 月の集客コスト：5万円

【質問】
- どの指標が業界平均より優れているか
- どこに改善余地があるか
- 業界トップ10%を目指すための優先課題
```

AIは業界知識をベースに**業界平均と自店の相対位置**を分析できる。

## SWOT分析のAI活用

```
私のジムのSWOT分析を作成してください。

【ジム情報】
- 立地：五反田駅徒歩5分
- 業態：パーソナル＋レンタルジム併用
- ターゲット：30〜50代女性
- 強み候補：AI食事管理、副業オーナー実体験
- 競合：5km圏内に大手チェーン3店

【出力】
- Strengths（強み）5個
- Weaknesses（弱み）5個
- Opportunities（機会）5個
- Threats（脅威）5個
- 戦略的優先課題3つ
```

## 予測・シミュレーション

```
今後12ヶ月の売上予測を、3シナリオでシミュレーションしてください。

【現状】
- 月商：50万円
- 月の純増顧客：1名
- 客単価：35,000円
- 継続率：85%/月

【シナリオ】
1. 楽観：純増+2名/月、客単価+5%/年
2. 標準：現状維持
3. 悲観：純増0名、退会率上昇

【出力】
- 各シナリオの月次予測（12ヶ月分）
- 累積収入
- 損益分岐点を超える時期
- リスク要因と対策
```

## AIの分析を「使う」コツ

### ✅ 効果的な使い方

#### 1. 具体的な数字を渡す
「ある程度」「だいたい」ではなく実数値を。

#### 2. 文脈を伝える
ターゲット・立地・業態を明示。

#### 3. 出力形式を指定
箇条書き・表・優先順位付きリスト。

#### 4. 複数回イテレーション
1回目の出力に追加質問・深掘り。

### ❌ 避けるべき使い方

- 漠然とした質問「どうすればいい？」
- データなしの相談
- AIの提案を鵜呑み（必ず吟味）
- 機密情報を直接入力（仮名化）

**月のAI活用：1.5時間**で経営の質が大きく上がる。

## 失敗パターン

### ❌ AIの提案を全部実行
リソースが足りない。優先順位をつけて1〜2個に絞る。

### ❌ データの集計を怠る
材料がなければAIも分析できない。

### ❌ 主観で判断し続ける
AIの客観的視点を活用しない。

### ❌ 単発で使う
継続的な活用で精度が上がる。

## まとめ

経営判断にAI（特にClaude）を活用すれば、**月1.5時間で「専門家のセカンドオピニオン」**を得られます。月次振り返り・価格改定・採用判断・集客戦略・SWOT分析・将来予測のすべてに使えます。AIを「優秀な参謀」として使い、「最終判断は自分」というスタンスで取り入れれば、経営の質と速度が劇的に上がります。
