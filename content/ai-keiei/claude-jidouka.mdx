# Claudeで業務を8割自動化する

副業でジムを運営する場合、**自分の稼働時間が最大の制約**です。AI（Claude）を活用すれば、これまで月20時間かかっていた業務を月4〜5時間に圧縮できます。本記事では実際に行っている自動化を公開します。

## 自動化できる業務リスト

| 業務 | 月の時間（手動） | AI活用後 | 削減率 |
|---|---|---|---|
| 集客記事執筆 | 15h | 2h | 87% |
| SNS投稿文作成 | 8h | 1h | 88% |
| 顧客向け食事アドバイス | 10h | 2h | 80% |
| 月次振り返り資料作成 | 4h | 30分 | 87% |
| メール・LINE返信文 | 5h | 1h | 80% |
| **合計** | **42h** | **6.5h** | **84%** |

**月35時間の節約**は、副業ジム経営者にとって生命線です。

## 1. ブログ記事の量産

### Step 1：キーワードリストを作る
以下をClaudeに依頼：

```
パーソナルジムの経営者が
ブログで書くべきSEOキーワードを30個提案してください。
検索ボリュームが期待でき、競合が少ないものを優先。
```

### Step 2：記事の構成を作らせる
各キーワードに対して：

```
キーワード「○○」で2,500文字のSEO記事を書きたい。
H2見出しを5つ提案してください。
読者は開業を検討中の個人パーソナルトレーナーです。
```

### Step 3：本文をドラフト
構成が決まったら本文を生成：

```
以下の構成で記事を書いてください。
- ターゲット：開業を検討中のトレーナー
- トーン：実体験ベース、専門用語は補足
- 文字数：2,500文字程度
- 構成：[Step 2で作った構成]
```

### Step 4：自分の体験を加筆
**ここが最重要**。AIが書いた記事に「自分の固有名詞・実数字・体験談」を必ず加える。これがないとSEO評価も読者の信頼も得られません。

## 2. SNS投稿の自動化

### 月1回のセッションで2週間分を生成

```
パーソナルジムのInstagram投稿を14本書いてください。

【条件】
- ターゲット：30〜50代女性、ダイエット目的
- 投稿パターン：以下を組み合わせ
  - 食事Tips 4本
  - トレーニング解説 4本
  - お客様の声 3本
  - 開業オーナーの想い 3本
- それぞれキャプション200〜300文字
- ハッシュタグも提案
```

これで**14日分の投稿原稿が10分で完成**。あとは画像と動画を撮るだけ。

## 3. 顧客の食事管理アドバイス

これが最も時間削減効果が大きい領域。

### 流れ
1. 顧客から「今日の食事写真」をLINEで受け取る
2. Claudeに以下のプロンプトで分析依頼：

```
以下の食事内容について、
ダイエット目的の40代女性向けに
100文字以内でアドバイスしてください。

【食事】
朝：トースト1枚、コーヒー
昼：ラーメン
夜：唐揚げ定食

【目標体重】−5kg
【現在の摂取目安】1,400kcal
```

3. AIの回答を確認して、必要なら微調整して送信

**1顧客あたり3分→30秒に短縮**。20顧客いても月1〜2時間で済みます。

## 4. 月次振り返り資料

```
以下の数字から、今月の振り返り資料を作ってください。

【売上】32万円（前月比+5万円）
【新規顧客】3名（前月比+1名）
【継続率】85%（前月比+3%）
【LP問い合わせ】15件（前月比+5件）

→ 良かった点・改善点・来月のアクションを
箇条書きで整理してください。
```

経営判断に必要な振り返りが**5分で完成**。意思決定の質も上がります。

## 5. メール・LINE返信のテンプレート化

### よくある問い合わせパターンを集めて回答テンプレを生成

```
パーソナルジムでよくある10の問い合わせと、
それぞれへの返信テンプレを作ってください。
丁寧かつ簡潔に。
```

これで返信時間が**1件5分→30秒**に。月20件の問い合わせがあれば、月1.5時間の節約。

## 6. 実際に使っているプロンプト集

### 体験トレーニング後のフォローアップLINE

```
体験トレーニングに来られた方への
24時間以内のフォローアップLINEを作ってください。

【条件】
- 押し売り感を出さない
- 体験で取り組んだ内容を振り返る
- 質問しやすい雰囲気を作る
- 末尾に予約への動線を入れる
```

### 月次のお客様向けメルマガ

```
パーソナルジム会員向けの月次メルマガを書いてください。

【今月のトピック】
- 季節に合わせたダイエットTips
- 新メニューの紹介
- お客様の成果報告（匿名OK）

【条件】
- 600〜800文字
- フレンドリーなトーン
- 末尾に紹介キャンペーンを入れる
```

## AIに任せきれない業務

すべてを自動化はできません。**「人」でないとできない業務**は以下：

- セッション本体（指導・モチベーション喚起）
- 顧客とのリアルな関係構築
- 経営判断・戦略決定
- トラブル対応

**「人がやるべき仕事に集中するため、AIで他を全部削る」**が正しい考え方です。

## 自動化を始める順番

1. **ブログ記事執筆**（最も時間削減効果が大きい）
2. **SNS投稿文作成**（次に効果的）
3. **顧客対応の食事管理**（収益への貢献も大）
4. **月次振り返り**（経営判断の質向上）
5. **返信テンプレ**（細かい時間の積み重ね）

**1ヶ月に1個ずつ自動化**を進めれば、半年で大半の業務がAI化できます。

## まとめ

Claudeを活用した業務自動化で、**月42時間→6.5時間（84%削減）**が実現可能です。空いた時間は新規集客・サービス改善・本業に集中できます。AIは「自分の代わりに作業する優秀な部下」と思って使うのが正解です。
