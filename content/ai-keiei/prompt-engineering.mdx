# AIを使いこなすプロンプトの基本

「ChatGPTもClaudeも使ってみたけど、思った答えが返ってこない」——これは**プロンプトの作り方の問題**。本記事では、ジム経営者が知っておくべきプロンプト設計の基本を実例で解説します。

## プロンプトの質が9割

| プロンプトの質 | AIの出力 |
|---|---|
| 雑（一文だけ） | 一般的・浅い回答 |
| 中程度 | 普通の回答 |
| **高品質（具体的）** | **専門家レベルの回答** |

**AIの能力を引き出せるかどうかは、プロンプト次第**。

## 基本構造：5つの要素

良いプロンプトには以下の5要素を含める：

1. **役割（誰として）**：「あなたは〇〇の専門家です」
2. **タスク（何を）**：「〇〇を作成してください」
3. **背景（なぜ）**：「〇〇のために必要です」
4. **条件（どのように）**：文字数・トーン・構成
5. **出力形式**：箇条書き・表・JSON等

## 悪いプロンプト vs 良いプロンプト

### ❌ 悪い例
```
ジムのインスタ投稿作って
```

### ✅ 良い例
```
あなたはパーソナルジム経営の専門家です。

【タスク】
Instagram投稿を5本作成してください。

【背景】
五反田の30〜50代女性向けパーソナルジムの集客のため。
今月のテーマは「夏に向けたボディメイク」。

【条件】
- 各投稿200〜300文字
- 絵文字を適切に使用
- ハッシュタグ20個（大3：中5：小5に分類）
- 押し売り感を出さず、価値提供重視

【出力形式】
投稿1: [本文]
ハッシュタグ: #...

投稿2: [本文]
...
```

**出力の質が10倍違う**。

## ジム経営で使える実践プロンプト集

### 1. ブログ記事の構成案

```
パーソナルジム経営者向けの
「日本政策金融公庫融資の通し方」記事の
構成案を作ってください。

【条件】
- H2見出しを6個
- 各見出しの下に詳細ポイント2〜3個
- 実数値・体験談を含めることを意識した構成
```

### 2. キャッチコピー

```
パーソナルジムのHPファーストビュー用に
キャッチコピーを10個作ってください。

【ターゲット】
30〜50代女性、ダイエット目的、五反田勤務

【条件】
- 各案30文字以内
- 「結果」「時間効率」「個別対応」のいずれかを訴求
- 競合と差別化された独自感
```

### 3. 顧客向けLINEメッセージ

```
パーソナルジムの新規顧客向けに
体験トレーニング翌日のフォローLINEを書いてください。

【背景】
体験で「腰痛が気になる」と言っていた40代女性。
入会を迷っている様子。

【条件】
- 200文字以内
- 押し売り感なし
- 質問しやすい雰囲気
- 末尾に予約ページのURLを入れる
```

### 4. 経営判断のサポート

```
パーソナルジム経営者として、以下の数字を分析し、
改善ポイントを5つ提案してください。

【7月の数字】
- 売上：480,000円（前月比+8%）
- 新規入会：3名
- 既存退会：1名
- 体験予約：12名
- 体験→入会率：25%（前月35%）
- 紹介経由：1名
- レンタルジム稼働率：30%

【着眼点】
- 体験→入会率の低下要因
- 紹介経由の少なさ
- レンタルジムの活用
```

### 5. 食事管理アドバイス

```
以下の食事内容について、
ダイエット目的の40代女性向けに
100文字以内でアドバイスしてください。

【食事】
朝：トースト1枚、コーヒー
昼：ラーメン
夜：唐揚げ定食

【目標】3ヶ月で5kg減量

【条件】
- 親しみやすく、ダメ出しではなく前向きに
- 具体的な改善案を1つ提示
```

## プロンプトのテクニック

### テクニック1：ロールプレイ

「あなたは〇〇です」で役割を明示。

```
あなたは20年の指導経験を持つNSCA認定パーソナルトレーナーです。
クライアントは産後8ヶ月の30代女性です。
腹筋を回復させるトレーニング計画を作ってください。
```

### テクニック2：例示（Few-shot）

例を見せることで、出力の方向性を制御。

```
次のスタイルでInstagram投稿を作ってください。

【お手本】
「お腹痩せの近道は腹筋じゃない！
実はヒップアップ系の種目が一番効きます。
ダイエット成功の8割は『どこを鍛えるか』で決まります。」

【新しい投稿テーマ】
「タンパク質摂取のタイミング」
```

### テクニック3：制約条件の明示

文字数・トーン・形式を細かく指定。

```
【文字数】200〜300文字
【トーン】親しみやすく、専門的すぎない
【口調】です・ます調
【含める要素】具体的な数字、体験談、CTA
```

### テクニック4：ステップバイステップ

複雑なタスクは段階に分ける。

```
以下の手順で進めてください：

Step 1: ターゲット顧客のペルソナを3行で定義
Step 2: そのペルソナの抱える悩みを5つ列挙
Step 3: 各悩みに対する解決策を提案
Step 4: 解決策を3つのキャッチコピーに圧縮
```

### テクニック5：出力フォーマット指定

JSON、表、Markdown等を明示。

```
出力は以下のMarkdown表で：

| 項目 | 内容 |
|---|---|
| ターゲット | ... |
| 強み | ... |
| 弱み | ... |
```

## プロンプトを改善する方法

### イテレーション（繰り返し改善）

1. 最初のプロンプトで出力を得る
2. 出力を確認し、足りない部分を特定
3. 「もっと〇〇に特化して」と追加指示
4. 満足するまで繰り返す

### 修正指示の例
- 「もっとカジュアルに書き直して」
- 「数字を具体的に入れて」
- 「文字数を300字以内に削って」
- 「専門用語を初心者向けに言い換えて」

## プロンプトテンプレート集

頻繁に使うプロンプトはテンプレ化してNotion等に保存：

### テンプレ1：ブログ記事生成用
```
あなたはパーソナルジム経営の専門家です。
{記事テーマ}について{文字数}文字の記事を書いてください。

【条件】
- 構成：H2見出し{個数}個
- トーン：{トーン}
- 含める要素：{要素}
```

### テンプレ2：SNS投稿生成用
```
{プラットフォーム}用の投稿を{本数}本作成。
ターゲット：{ターゲット}
テーマ：{テーマ}
各投稿{文字数}文字、ハッシュタグ{個数}個。
```

## やってはいけないプロンプト

### ❌ 曖昧すぎる
「いい感じに書いて」「良いインスタ投稿作って」

### ❌ 矛盾する指示
「短くしっかり詳しく」（短いか詳しいか選ぶ必要）

### ❌ 範囲を限定しない
「日本人向けに」だけでは不十分。年齢・性別・目的まで指定。

### ❌ 倫理的に問題のある依頼
「絶対に痩せる」「医療行為レベルのアドバイス」など。

**頻繁に使うプロンプトはテンプレ化して再利用**するのが効率化のコツ。

## まとめ

AIを使いこなす鍵は**プロンプト設計**にあります。「役割・タスク・背景・条件・出力形式」の5要素を意識して具体的に書けば、AIは専門家レベルの回答を返してくれます。よく使うプロンプトはテンプレ化し、改善を繰り返すことで、月の業務時間を大幅に削減できます。
