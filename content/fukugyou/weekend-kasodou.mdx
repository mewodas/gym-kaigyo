# 週末・夜間だけで月10万円を稼ぐ副業術

「平日昼は本業、副業は限られた時間で」——多くの副業トレーナーが直面する制約です。本記事では、週末と平日夜間だけで月10万円以上を稼ぐ具体的な方法を解説します。

## 副業の現実的な稼働時間

### 会社員副業の典型パターン

| 時間帯 | 稼働可能性 | 推奨度 |
|---|---|---|
| 平日朝（6〜8時） | △ | ★ |
| 平日昼（12〜13時） | × | × |
| 平日夜（19〜22時） | ○ | ★★★ |
| 土曜（終日） | ◎ | ★★★ |
| 日曜（終日） | ◎ | ★★ |

**週12〜20時間**が現実的な稼働時間。

## 月10万円達成の数式

```
セッション単価8,000円 × 月13セッション = 104,000円
レンタルジム代 1,650円 × 13回 = -21,450円
実利益 = 約82,000円
```

または：

```
セッション単価10,000円 × 月10セッション = 100,000円
レンタルジム代 = -16,500円
実利益 = 約83,500円
```

**月10〜13セッションが目標**。

## 顧客獲得の現実

### 月10セッションを実現するクライアント数

| パターン | 必要顧客数 |
|---|---|
| 全員週1回 | 2〜3名 |
| 月3〜4回ペース | 3名 |
| 月2回ペース | 5名 |
| 月1回ペース（混在） | 5〜10名 |

**最低3名の継続顧客**が確保できれば月10万円が見える。

## 効率的な週末スケジュール

### 土曜日のモデル例

| 時間 | 活動 |
|---|---|
| 8:00 | 起床・準備 |
| 9:00〜10:00 | 顧客A：パーソナル |
| 10:30〜11:30 | 顧客B：パーソナル |
| 12:00〜13:00 | 休憩・移動 |
| 13:00〜14:00 | 顧客C：パーソナル |
| 14:30〜15:30 | 顧客D：パーソナル |
| 16:00〜17:00 | 顧客E：パーソナル |
| 18:00〜 | 自分のリフレッシュ |

**土曜1日で5セッション**。月4回で20セッション可能。

### 平日夜のモデル例

| 時間 | 活動 |
|---|---|
| 19:30 | 本業終了・移動 |
| 20:00〜21:00 | 顧客F：パーソナル |
| 21:30〜22:30 | 顧客G：パーソナル（必要時） |

**平日週2回で4セッション**。月4セッション。

### 合計
- 土曜：20セッション/月
- 日曜：8〜10セッション/月（本人の余力次第）
- 平日夜：4〜8セッション/月
- **月合計：30〜40セッション可能**

ただし**疲労管理**を考えると月15〜20セッションが現実的なライン。

## 時間効率を最大化する5つの工夫

### 工夫1：60分→45分セッションへ

#### メリット
- 同じ時間で多くの顧客を担当
- 顧客の負担も軽減
- 短時間集中で効果的

#### 設計のコツ
- 5分：ウォームアップ
- 35分：メイントレーニング（高密度）
- 5分：クールダウン

### 工夫2：連続セッションの組み方

#### NGパターン
```
9:00〜10:00 顧客A
11:30〜12:30 顧客B（1.5時間空く）
14:00〜15:00 顧客C
```

#### OKパターン
```
9:00〜10:00 顧客A
10:30〜11:30 顧客B（30分インターバル）
12:00〜13:00 顧客C
```

**インターバル30分で連続化**することで時間効率UP。

### 工夫3：固定スケジュールへの誘導

「毎週土曜10時」のような**固定枠**で予約してもらう仕組み。

#### メリット
- 予約管理が楽
- 顧客も習慣化しやすい
- 無断キャンセル減少

### 工夫4：オンラインセッション併用

#### メリット
- 移動時間ゼロ
- 平日夜の自宅からも対応可
- 雨・体調不良時の代替案

#### 限界
- 全セッションをオンライン化はできない
- 設備・環境の問題

**月の20%程度をオンラインに**するのが現実的。

### 工夫5：顧客の「平日夜」枠を確保

平日夜を希望する顧客に割増料金を設定すると：
- 8,000円 → 10,000円（25%UP）
- 月4セッションで売上+8,000円
- 自分の本業終了後の貴重な時間に値段をつける

セッション以外に**月12時間程度**は集客に時間を使う。

## 燃え尽き防止の重要性

### 副業の落とし穴
- 本業で疲れている上に副業
- 休日も働き続ける
- リフレッシュの時間がなくなる
- パートナー・家族との時間減少

### 防止策
1. **月1回の完全休息日**を確保
2. **稼働時間の上限**を決める（例：月60時間まで）
3. **本業との優先順位**を明確化
4. **家族の理解**を得る

**1〜2ヶ月で燃え尽きないペース設計**が長期的成功の鍵。

## レンタルジムの効率的な使い方

### 使い分けのパターン

| 時間帯 | レンタルジム |
|---|---|
| 土曜午前 | 一括予約（1日借り） |
| 土曜午後 | 同上 |
| 日曜 | 必要に応じて |
| 平日夜 | 時間貸し |

### 月額パックの活用

- 月20時間パック：25,000円
- 1時間あたり：1,250円
- **時間貸しより25%安い**

## 失敗パターン

### ❌ 本業をおろそかにする
副業がメインになると本業から信頼を失う。

### ❌ 価格を安くしすぎる
時給3,000円相当だとレンタル料引いて時給1,500円。

### ❌ スケジュールを詰めすぎる
燃え尽きで突然続けられなくなる。

### ❌ 確定申告を怠る
年20万円超は申告必須。

## まとめ

週末＋平日夜だけで月10万円を稼ぐには**「3名の継続顧客 × 月3〜4セッション」**が最短ルート。45分セッション・連続予約・固定スケジュールで時間効率を最大化、月1回の完全休息日で燃え尽きを防げば、長期的に持続可能な副業として成り立ちます。
