# ジムに最適な決済システムを選ぶ

「現金だけでは不便」「カード決済を入れたい」——個人ジム経営者が必ず検討する課題。本記事では、Square・Stripe・PayPalなど主要決済サービスを比較し、ジムに最適な選択肢を解説します。

## なぜ決済システムが必要か

| 決済方法 | 顧客の利便性 | 売上機会 |
|---|---|---|
| 現金のみ | 低 | 機会損失大 |
| 銀行振込のみ | 中 | 即時決済できない |
| **クレジット対応** | **高** | **増加** |
| サブスク決済対応 | 最高 | 継続率も上がる |

**カード決済できないだけで月3〜5万円の機会損失**が発生することも。

## 主要決済サービス比較

| サービス | 月額 | 決済手数料 | サブスク対応 | おすすめ度 |
|---|---|---|---|---|
| **Square** | 無料 | 3.25% | あり | ★★★★★ |
| Stripe | 無料 | 3.6% | あり | ★★★★ |
| PayPal | 無料 | 3.6% + 40円 | あり | ★★★ |
| Airペイ | 無料 | 3.24% | なし | ★★★ |
| stera pack | 月3,300円 | 2.7% | なし | ★★★ |

## 1. Square（おすすめ度：★★★★★）

### 概要
- 米国発、世界最大級の決済プラットフォーム
- 個人事業主にも使いやすい

### 決済手数料

| 決済手段 | 手数料 |
|---|---|
| クレジットカード（VISA/Master） | 3.25% |
| クレジットカード（JCB/AMEX） | 3.95% |
| QRコード決済 | 3.25% |
| 電子マネー | 3.25% |

### 提供サービス

| サービス | 用途 |
|---|---|
| Square リーダー | カード決済端末 |
| Square ターミナル | レジとして使える |
| Square Online Store | ネットショップ |
| Square 請求書 | 請求書発行 |
| Square Appointments | 予約管理 |
| Square サブスクリプション | 月額課金 |

### メリット
- **入金が速い**（最短翌営業日）
- **追加手数料なし**（月額無料）
- **多機能**（予約・在庫・請求書）
- **使いやすいUI**

### デメリット
- 手数料がやや高め
- AMEX・JCBの手数料が割高

### こんなジムに：
- 個人〜小規模ジム
- 対面・オンライン両方使う
- 多機能を重視

## 2. Stripe（おすすめ度：★★★★）

### 概要
- IT・スタートアップ向け
- API経由のカスタマイズに強い

### 決済手数料
- 国内クレジット：3.6%
- 海外カード：3.6% + 円換算手数料

### メリット
- **API・カスタマイズが超強力**
- サブスク決済の自由度が高い
- 多通貨対応
- セキュリティ最高レベル

### デメリット
- 設定にエンジニア知識が必要
- 物理的な決済端末がない
- サポートは英語ベース（日本語対応も改善中）

### こんなジムに：
- HPでオンライン決済をやりたい
- サブスクの細かい設計をしたい
- 開発者がいる or 自分で設定できる

## 3. PayPal（おすすめ度：★★★）

### 概要
- 世界的に有名な決済サービス
- 個人間送金にも使える

### 決済手数料
- 国内：3.6% + 40円
- 海外：4.1% + 40円

### メリット
- **海外顧客にも対応しやすい**
- 知名度が高い
- アカウント間で簡単に送金

### デメリット
- 手数料がやや高い
- 国内ジムでの利用率は低い
- 対面決済には不向き

### こんなジムに：
- 海外からのオンラインセッション提供
- 既にPayPalアカウントがある人

## 4. Airペイ（おすすめ度：★★★）

### 概要
- リクルート提供
- 国内の中小事業者に強い

### 決済手数料
- クレジット：3.24%
- 電子マネー：3.24%

### メリット
- **国内対応が手厚い**
- 入金サイクルが早い
- リクルート系（ホットペッパー等）と連携

### デメリット
- サブスク決済機能なし
- 機能はSquareより少ない

### こんなジムに：
- 都度払いがメインのジム
- リクルート系のシステムを使っている

## 5. stera pack（おすすめ度：★★★）

### 概要
- SMBC・GMO・ビザなどの共同サービス
- 中規模事業者向け

### 決済手数料
- クレジット：2.7%（最安級）
- 月額：3,300円

### メリット
- **手数料が業界最安レベル**
- マルチ決済対応
- 銀行系の信頼性

### デメリット
- 月額が必要
- 中規模以上向き
- 個人事業主は審査がやや厳しい

### こんなジムに：
- 月の決済額が大きい（150万円以上）
- 手数料を限界まで抑えたい

## サブスク（月額課金）対応の重要性

### サブスク化のメリット

| 項目 | 効果 |
|---|---|
| 顧客の解約ハードル | 高くなる（継続率UP） |
| 売上の予測 | しやすい |
| 入金サイクル | 安定 |
| 経理処理 | 自動化可能 |

**月額固定プランの導入で月の売上が20〜30%安定化**。

### サブスク対応サービス

| サービス | 設定の容易さ | 機能 |
|---|---|---|
| Square サブスクリプション | ★★★ | 標準機能 |
| Stripe Billing | ★ | 高度なカスタマイズ可 |
| PayPal Subscriptions | ★★ | シンプル |

## 決済システム選定の判断フローチャート

```
Q1：月の決済額は150万円以上？
├ Yes → stera pack（手数料最安）
└ No → Q2へ

Q2：オンライン決済も必要？
├ Yes → Q3へ
└ No → Airペイ または Square

Q3：開発・カスタマイズ可能？
├ Yes → Stripe
└ No → Square
```

**個人ジムの大半は Square で十分**。

## 設置・契約の流れ

### Squareの場合（最速）

1. **アカウント開設**：5分
2. **本人確認・銀行口座登録**：1〜3日
3. **カードリーダー注文**：3,500円〜
4. **設置・テスト**：当日〜翌日
5. **運用開始**

**最短1週間で導入可能**。

## 顧客への案内

### 決済方法のメニュー化

HPで明示：
> ご利用可能な決済方法：
> ・現金
> ・クレジットカード（VISA・Master・JCB・AMEX）
> ・QRコード決済（PayPay・LINE Pay・楽天Pay）
> ・電子マネー（Suica・PASMO・iD・QUICPay）
> ・銀行振込

**選択肢の多さは安心感**につながる。

## 失敗パターン

### ❌ 手数料の高さに気づかず
3.6% × 月の決済額 = 隠れたコスト。**安いサービスへの乗り換え検討**。

### ❌ サブスク機能がない
都度の請求作業に時間取られる。

### ❌ 入金サイクルが遅い
キャッシュフロー悪化。**翌営業日入金**を選ぶ。

### ❌ 決済端末を導入しない
カード決済できないだけで離脱が発生。

## 経理連携の重要性

### freeeとの連携

Square・Stripeは**freeeに自動連携**できる：
- 売上が自動で記録
- 仕訳の手間ゼロ
- 月次決算が高速

**経理時間が月5時間→1時間**に短縮可能。

## まとめ

ジム向け決済システムは**Squareが最もバランス良い選択**。月額無料・多機能・使いやすさで個人ジムには最適。月150万円超ならstera packで手数料節約も検討。サブスク機能は必須、freee等の経理連携を選ぶことで業務効率も大きく改善します。
