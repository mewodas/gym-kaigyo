# LINE公式アカウントこそジム集客の決定打

LINE公式アカウントは「**SNS・Googleで興味を持った人を確実にクロージングする装置**」。本記事では設定から運用までの実践方法を解説します。

## なぜLINE公式アカウントが重要なのか

| チャネル | 開封率 | 即時性 |
|---|---|---|
| Eメール | 10〜20% | 低 |
| Instagram DM | 30〜40% | 中 |
| **LINE公式** | **60〜80%** | **高** |

**LINEは日本人の95%が使用**。問い合わせ→入会までの流れが圧倒的にスムーズ。

## 開設から運用までの全体像

```
1. アカウント開設（30分）
2. プロフィール・基本設定
3. リッチメニュー作成
4. ステップ配信（自動メッセージ）設計
5. 既存顧客・新規へ告知
6. 日々の運用
```

## アカウント開設手順

### Step 1：LINE公式アカウントを作成

1. [LINE for Business](https://www.linebiz.com/jp/) にアクセス
2. 「無料で始める」→アカウント作成
3. ジム名・連絡先・カテゴリを入力

### Step 2：プラン選択

| プラン | 月額 | メッセージ送信数 |
|---|---|---|
| コミュニケーションプラン | 無料 | 月200通 |
| ライトプラン | 5,000円 | 月5,000通 |
| スタンダードプラン | 15,000円 | 月30,000通 |

**最初は無料プランで十分**。会員50名以下ならまず収まります。

## プロフィール最適化

### 必須要素

- **アイコン**：ジムロゴ or 代表者の顔写真
- **カバー画像**：ジムの内観 or トレーナー
- **挨拶文**：登録時に自動送信されるメッセージ

### 挨拶文の例

```
🏋️ ジム開業ラボ公式LINEへようこそ！

✅ 開業相談、AI活用のご質問、お気軽に
✅ 体験予約も可能です

【メニュー】
👇下のボタンからお選びください👇
```

## リッチメニューの設計

リッチメニューは**LINE画面下部の固定メニュー**。問い合わせの導線をここに集約します。

### 基本構成（6マス構成）

| 配置 | コンテンツ |
|---|---|
| 左上 | 体験予約 |
| 中上 | 料金・プラン |
| 右上 | アクセス・営業時間 |
| 左下 | お客様の声 |
| 中下 | よくある質問 |
| 右下 | 友だちを紹介する |

### デザインのコツ

- **アイコンを統一**（同じトーンの色味・形）
- **文字は大きく**（スマホで読みやすく）
- **クリック率の高いものを左上に**

### 制作ツール
- Canva（無料・テンプレあり）
- LINE公式の標準ツール（簡易）
- 外注（Coconalaで5,000〜15,000円）

## ステップ配信の設計

ステップ配信は**登録後に自動で送られるシリーズメッセージ**。離脱を防ぎ、入会率を上げます。

### 5ステップ配信の例

#### Day 1（登録直後）：歓迎メッセージ
```
ご登録ありがとうございます！

このアカウントでは
✅ ジム開業のリアル
✅ AI活用の事例
✅ 副業から独立への道
を発信しています。

まずはこちらの記事をどうぞ👇
[記事リンク]
```

#### Day 2：信頼構築コンテンツ
```
昨日の続きです！

実際に開業してわかった
「物件選びで失敗する5パターン」を
こちらの記事で解説しています👇

[記事リンク]
```

#### Day 3：価値提供
```
今日は無料プレゼントです🎁

「開業前チェックリスト30項目」を
PDFで配布中。下記からダウンロード👇

[PDFリンク]
```

#### Day 5：さらなる価値
```
お客様からよく聞かれる
「補助金の使い方」について
動画で解説しました📹

[YouTubeリンク]
```

#### Day 7：オファー
```
1週間お付き合いいただきありがとうございます！

もしご興味があれば
「無料相談（30分）」を実施中です。

[予約フォーム]
```

**重要：オファーは7日目以降に**。早すぎると逃げられます。

## 配信のベストプラクティス

### 配信頻度
- **登録初週**：3〜5回（ステップ配信）
- **その後**：週1回の一斉配信
- **多すぎはブロック原因**：月10通超は注意

### 配信時間帯
- ターゲットが見やすい時間帯を選ぶ
- 主婦層なら平日10〜15時
- 会社員なら平日19〜21時 or 土日午前

### コンテンツ形式

| 形式 | エンゲージメント | 配信通数（無料プラン） |
|---|---|---|
| テキストのみ | 中 | 1通分 |
| 画像付き | 高 | 1通分 |
| カルーセル | 高 | 1通分 |
| 動画 | 高 | 1通分 |
| クーポン | 最高 | 1通分 |

## 既存顧客にLINEへ移行してもらう

### 移行のステップ

1. **告知**：店内ポスター・LINE登録特典の案内
2. **特典提示**：「LINE登録でセッション5%OFF」など
3. **登録誘導**：QRコードを店内・名刺に印刷
4. **個別案内**：セッション中に1人ずつ案内

**目標：既存顧客の80%以上をLINE登録**へ。

## 個別チャットの活用

### よくある問い合わせと対応

| 問い合わせ | 対応 |
|---|---|
| 体験予約 | 予約フォームのリンクを送信 |
| 料金質問 | リッチメニューを案内 |
| 雑談・励まし | 自動返信ではなく手動で返信（信頼形成） |
| キャンセル | キャンセルポリシーを共有 |

**24時間以内の返信が基本**。

## 友だち追加を増やす施策

### 1. SNSプロフィールにLINE追加URL
Instagram・Xのプロフィールに追加URLを記載。

### 2. ホームページのCTAをLINEに統一
「お問い合わせ」をLINE登録ボタンに集約。

### 3. 店内QRコード
受付・トイレ・トレーニングエリアに設置。

### 4. 紹介特典
LINE経由の紹介で両者にプレゼント。

| 指標 | 目標値（半年後） |
|---|---|
| 友だち数 | 100〜300人 |
| ブロック率 | 10%以下 |
| 開封率 | 50%以上 |
| 応答率（チャット） | 90%以上 |
| LINE経由の入会数 | 月3〜10人 |

## 失敗パターン

### ❌ 配信が一方通行
販促ばかりだとブロックされる。**価値提供：販促 = 8:2**を意識。

### ❌ ステップ配信を作っていない
登録直後の盛り上がりを活かせない。

### ❌ リッチメニューが分かりにくい
タップ先が不明確だと離脱。

### ❌ 個別チャットへの返信が遅い
24時間以上の放置はNG。

## まとめ

LINE公式アカウントは**ジム集客のクロージング装置**。リッチメニュー・ステップ配信・個別チャットの3点セットで運用すれば、SNS・SEOからの流入を確実に入会につなげられます。**月10〜30名の新規入会**が現実的なゴール。今日から運用を始めましょう。
