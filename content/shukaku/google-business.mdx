# Googleビジネスプロフィールでジムを満員にする

「パーソナルジム ＋ 地域名」で検索したときに表示される地図上の店舗一覧——これが**MEO（マップエンジン最適化）**です。**個人ジムにとってMEOは最も費用対効果の高い集客手段**で、口コミ50件を超えると毎週の問い合わせが安定します。

## なぜGoogleビジネスプロフィールが最強なのか

| 項目 | 効果 |
|---|---|
| 費用 | 完全無料 |
| 上位表示までの期間 | 3〜6ヶ月 |
| 想定流入数 | 月50〜200件（地域・カテゴリにより変動） |
| 問い合わせ転換率 | 5〜15% |
| 客の購買意図 | **極めて高い**（「今すぐ通いたい」層） |

Instagram・TikTokは「認知拡大」が主目的ですが、**Googleマップ検索者は「すぐに通うジムを探している」**ため、転換率が圧倒的に高いです。

## 初期設定（30分で完了）

### Step 1：オーナー登録

1. [Googleビジネスプロフィール](https://www.google.com/business/) にアクセス
2. 「ビジネスを管理」→「ビジネスを追加」
3. 店舗情報を入力（店名・住所・電話・カテゴリ）
4. オーナー確認（ハガキで認証コードが届く・1〜2週間）

### Step 2：基本情報を完璧に埋める

- **ビジネス名**：店舗の正式名称（キーワードを無理に入れない）
- **カテゴリ**：「パーソナルトレーニングジム」を選択（複数登録可）
- **営業時間**：祝日も含めて正確に
- **電話番号**：携帯でもOK
- **ウェブサイト**：あれば必ず登録
- **属性**：「予約必須」「Wi-Fiあり」「クレジットカード可」など該当するもの全て

## 写真の重要性（最優先施策）

**写真の有無で来店確率が3〜5倍変わる**と言われています。

### 必須の写真カテゴリ

| カテゴリ | 推奨枚数 | 内容 |
|---|---|---|
| 外観 | 5〜10枚 | 入口・看板・ビル全体 |
| 内観 | 10〜20枚 | トレーニングエリア全体・各器具 |
| トレーナー | 3〜5枚 | プロフィール写真・指導風景 |
| ロゴ・カバー | 各1枚 | ブランドビジュアル |
| 動画 | 3〜5本 | 60秒以内のジム紹介 |

**月1〜2枚のペースで新規追加**するとアルゴリズムに評価されやすくなります。

## 口コミの集め方（最重要KPI）

口コミ件数は順位に直接影響します。**口コミ50件超えると地図上位3位以内**が見えてきます。

### 口コミ依頼の3ステップ

1. **セッション後の満足度確認**
   「今日のセッションいかがでしたか？」と聞く

2. **満足度が高い客に依頼**
   「もしよろしければGoogleマップに口コミをいただけませんか？他のお客様の参考になります」

3. **QRコード化したリンクを送る**
   GoogleビジネスプロフィールのURLをQRコード生成→LINEで送付

### 口コミ依頼の文例（LINE送信）

> ◯◯様
> 本日もお疲れさまでした！
> もし差し支えなければ、Googleマップへの口コミをご投稿いただけますと大変ありがたいです。
> ［URLまたはQRコード］
> 引き続きよろしくお願いいたします！

**満足度の高い顧客の50%は協力してくれる**のが体感値です。

### NGな口コミ集めパターン

- **報酬と引き換えに依頼する**：規約違反、ペナルティ対象
- **業者から購入する**：すぐにバレる、アカウント停止
- **同一IPから複数口コミ投稿**：不自然と判定される

## 投稿機能をフル活用

Googleビジネスプロフィールには「投稿」機能があり、最新情報を表示できます。

### 投稿パターン

- **Before/After 報告**（顧客の許可を取って）
- **新メニュー・キャンペーン情報**
- **トレーナーの豆知識・栄養Tips**
- **季節のお知らせ**

**週1〜2回の投稿**で「アクティブな店舗」と評価されます。

## 検索クエリの最適化

「パーソナルジム ＋ 地域名」だけでなく、以下のキーワードでも上位表示を狙います。

- **「ダイエット ジム ＋ 地域名」**
- **「パーソナルトレーニング ＋ 駅名」**
- **「女性専用ジム ＋ 地域名」**（該当する場合）
- **「初心者向け ジム ＋ 地域名」**

ビジネス説明文・投稿・口コミにこれらのキーワードが自然に含まれていると順位が上がります。

## 競合分析で優位に立つ

地域の競合上位3店舗をリサーチします。

| 確認項目 | 自店との比較 |
|---|---|
| 口コミ件数 | 何件多い/少ないか |
| 平均評価 | 4.5以上が標準 |
| 写真枚数 | 30枚以上が望ましい |
| 投稿頻度 | 週何回か |
| 営業時間 | 競合より長くするか |

**自店が口コミ件数で2位になれれば、十分上位表示が見える**位置に来ます。
