# X投稿生成プロンプト

記事を書いたら、このプロンプトでXの投稿草稿を一括生成する。生成結果は `pipeline/x-posts/{slug}.md` に保存。

---

## 自動実行手順

1. `content/{category}/{slug}.mdx` を読む
2. 以下のプロンプトテンプレートに当てはめてClaudeで生成
3. 出力を `pipeline/x-posts/{slug}.md` に保存
4. `topics.json` の該当項目に `xGenerated: true` を追加

---

## マスタープロンプト

```
以下のジム開業ラボの記事から、X（旧Twitter）の投稿草稿を生成してください。

【記事】
{記事のMDXコンテンツをここに貼る}

【生成して欲しいもの】

A. ノウハウスレッド（6〜8ツイート）
   - ツイート1：強烈なフック＋スレッド予告（「◯◯の話」「以下スレッド」）
   - ツイート2〜N：本文の要点を1ツイートずつ
   - 最終ツイート：記事URLへのCTA

B. 単発ポスト（3本）
   - 1本目：記事のキーメッセージを140字で
   - 2本目：実数値・失敗談を切り出して
   - 3本目：質問・問いかけ形式で

C. 告知ツイート（1本）
   - 記事タイトル＋3つの要点＋URL

【ルール】
- 各ツイート140字以内（厳守）
- 絵文字は1ツイートに2〜3個まで
- ハッシュタグは投稿ごとに最大3個
- 「実体験」「数字」「具体例」を必ず含める
- 押し売り感のない自然な文体
- 「副業からスタート」「2業態運営」「AI活用」のうち1つは含める

【ハッシュタグ候補】
#副業 #ジム経営 #パーソナルトレーナー #AI活用 #フィットネス業界 #Claude #副業トレーナー

【URL】
記事URL：https://gym-kaigyo.jp/{category}/{slug}

【出力フォーマット】
## A. ノウハウスレッド

### ツイート1
[内容]

### ツイート2
[内容]

...

## B. 単発ポスト

### Post 1
[内容]

### Post 2
[内容]

### Post 3
[内容]

## C. 告知ツイート

[内容]
```

## 例：物件選び記事から生成した投稿

### A. ノウハウスレッド

**ツイート1**
```
パーソナルジム開業で
失敗する人の8割は「物件選び」で間違える。

副業からジムを開業して気づいた
物件選定の3軸を整理した🧵

#ジム経営 #副業
```

**ツイート2**
```
1️⃣ 立地

「徒歩7分以内」が最低ライン。
徒歩10分超は車・バイクで通う層しか取れず、
商圏が極端に狭まる。

家賃が安くても客が来なきゃ意味なし。
```

**ツイート3**
```
2️⃣ 広さ

パーソナル単業態なら12〜15坪で十分。
20坪超えは家賃負担が重すぎる。

筆者は2業態（PT＋レンタル）運営なので20坪確保。
業態数で逆算して決めるべき。
```

**ツイート4**
```
3️⃣ 賃料

月商の15〜20%が許容ライン。
これを超えると「働いても家賃で消える」
構造になる。

月商60万円なら家賃12万円まで。
シビアに見るほど後で楽になる。
```

**ツイート5**
```
落とし穴：居抜き物件

魅力的に見えるが、前テナント廃業の理由が
「立地が悪い」可能性大。

✅ 床の補強
✅ 換気・空調
✅ 防音
✅ 電気容量
✅ 給排水

最低この5点は内見時に必ず確認。
```

**ツイート6**
```
契約時の罠

✅ 解約予告期間（3〜6ヶ月が標準）
✅ 原状回復の範囲
✅ 看板設置可否

長すぎる解約予告期間の物件は避ける。
撤退時のコスト100万超もある。
```

**ツイート7**
```
結論：

「ここしかない」と思った時が最も危険。

最低5物件は内見してから判断する。
急ぐほど痛い目を見るのが物件選び。

詳しくは記事にまとめました↓
gym-kaigyo.jp/kaigyo/bukken-erabi
```

### B. 単発ポスト

**Post 1**
```
副業ジム経営者として断言する。

物件選びは開業の8割を決める。
立地・広さ・賃料の3軸で必ず複数比較。

「ここしかない」と思った時が
一番危険なサイン。
```

**Post 2**
```
居抜きで200万円損した話。

格安だと飛びついた居抜きジムを契約
→ 前店舗の原状回復義務を引き継いでいた
→ 撤退時に修繕費200万円請求

契約書の「原状回復範囲」は必ず確認する。
```

**Post 3**
```
パーソナルジム開業を考えてる方に質問。

物件選びで一番気になっているのは？

🅰️ 家賃
🅱️ 立地
🅲️ 広さ
🅳️ 設備（防音・電気）

リプで教えてください。
具体的なアドバイス返します。
```

### C. 告知ツイート

```
🆕 新記事公開

「パーソナルジム物件選びの完全ガイド｜失敗しない3つの軸」

▼この記事でわかること
・物件選定の3軸（立地・広さ・賃料）
・居抜き物件の見極め方
・契約で見落としがちな5つの罠

副業から実際にジムを開業した実体験で書きました。

gym-kaigyo.jp/kaigyo/bukken-erabi

#ジム経営 #副業 #パーソナルトレーナー
```

## 投稿運用フロー

### 週1回（推奨：日曜夜）

1. 過去1週間に書いた記事の `pipeline/x-posts/{slug}.md` を確認
2. ツイートを **TweetDeck** または **X Pro** の予約投稿機能に登録
3. 投稿時間を `x-strategy.md` の推奨スケジュールに従って分散
4. 1週間分が予約完了したら終了

**所要時間：30分／週**

### 1ヶ月目だけは手動投稿でリズムを掴む
最初の1ヶ月は予約せずに手動で投稿し、反応を見ながら型を作る。2ヶ月目から予約投稿に切り替えるのが学習効率良い。
